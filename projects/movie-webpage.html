Movie Webpage project

Context

My project was to create a movie webpage with my partner and using my skills of basic CSS and HTML to create a website trailer or showcase of the movie

This project required me to do a few steps in order for me get full credit:

  1. I first needed to create a plan.md file to plan out the content of the movie, planning out the different elements, properties, and selectiors and who would do what parts for the project.
  2. Next, I needed to implement the plannings I had for my HTML and CSS and place it onto my code.
  3. lastly I needed to complete a form to submit all of my work and get a final grade.

The challenge I faced

I was struggling with making the radio buttons so that only one could be clicked.

At first I tried to change the name="" attribute to name="data" so that the radio buttons inside the label tags could only click one radio button at a time but that didnt work

So... I checked my radio elements in my html and I realized that there was alot of spelling errors in my id attributes.

I quickly made sure to change all my form elements and tags so that the buttons could be clicked on seperately and it worked!

takeaways

overall, this project taught me the importance of always double checking and organizing your work before you end up submitting.

If I didnt take the time to check over the spelling errors that I had made for the different id elements then I would have never been able to solve that problem
